#ENGLISH

## Portfolio V1.0.0 - Luis Roca

Personal portfolio designed with a minimalist approach and performance.

### Technologies
- Semantic HTML5
- Tailwind CSS (v3.4) via CDN
- Vanilla JavaScript (ES6+)
- Formspree for email management

### Local Installation
1. Clone the repository.

2. Open `index.html` in your browser.
